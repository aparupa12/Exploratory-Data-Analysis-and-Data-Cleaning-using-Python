# Exploratory Data Analysis (EDA) on Sales Dataset

# Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset to uncover patterns, trends, and insights. The analysis was carried out using Python, Pandas, and Matplotlib.

# Objective
- Understand dataset structure and variables
- Analyze sales trends using TotalPrice
- Study Quantity distribution
- Identify patterns and possible outliers

# Methodology
1. Data Loading  
   - Imported Excel dataset using Pandas  

2. Data Understanding  
   - Checked dataset info and column structure  
   - Reviewed summary statistics  

3. Statistical Analysis  
   - Calculated mean, median, and count for TotalPrice  
   - Analyzed product-level variations  

4. Visualization  
   - Created histogram to analyze Quantity distribution
